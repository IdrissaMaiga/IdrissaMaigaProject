## Project Information

- **Project Name**: Product Assistant - AI-Powered Shopping Experience
- **Description**: A comprehensive .NET MAUI mobile application that provides a unified conversational shopping experience with AI-powered product search and recommendations.
- **Technologies**: .NET MAUI, C#, ASP.NET Core, Google Gemini AI, SQLite, Docker, Kubernetes
- **License**: Educational/Academic Project

## Acknowledgments

This project was developed as a semester assignment for the .NET MAUI course, demonstrating:
- MVVM architecture implementation
- Microservices architecture with Docker and Kubernetes
- AI integration with Google Gemini API
- Mobile development with .NET MAUI
- Web scraping capabilities
- Authentication with Auth0
- SQLite database for data persistence
